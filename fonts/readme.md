Line Spacing:
- [x] LGS = Line Gap Small
- [ ] LGM = Line Gap Medium
- [ ] LGL = Line Gap Large

DZ Suffix:
- [x] DZ = "Dotted Zero" - the zero character has a dot/slash through it to distinguish from letter O
- [ ] No DZ = Regular zero

Font Width:
- [x] NerdFont (no suffix) = Regular/standard width
- [ ] NerdFontMono = Monospaced variant (all characters same width)
- [ ] NerdFontPropo = Proportional variant
